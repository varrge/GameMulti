# 项目 Roadmap

## 阶段 0：规划与技术定稿

### 目标
把项目边界、技术选型、数据库模型、接口协议和任务拆分全部固定下来。

### 关键产出
- 项目总体规划
- 技术选型文档
- 数据库设计与字段解释
- API 边界文档
- OpenMOSS 任务拆分文档

---

## 阶段 1：邀请制账户与绑定 MVP

### 目标
跑通邀请制注册与游戏内发起绑定。

### 范围
- 邀请码注册
- 主站登录
- 绑定会话
- 链接/配对码确认
- 用户中心显示绑定结果

### 里程碑结果
玩家可以从游戏内发起绑定，并在主站完成账户绑定确认。

---

## 阶段 2：首个 PoC 游戏接入

### 目标
接通首个 PoC 游戏插件。

### 范围
- 插件鉴权
- 绑定命令
- 事件上报
- 状态上报
- 回执机制

### 里程碑结果
主站可与首个游戏服务器稳定通信。

---

## 阶段 3：金币与商城闭环

### 目标
让玩家行为开始转成金币，并支持消费金币。

### 范围
- 奖励规则
- 事件结算
- 钱包与流水
- 商城与订单
- 发货任务

### 里程碑结果
形成“玩游戏 -> 得金币 -> 花金币”的完整闭环。

---

## 阶段 4：论坛与后台运营闭环

### 目标
补齐论坛打通和后台运营能力。

### 范围
- Discourse SSO
- 论坛用户同步
- Admin 面板
- 邀请码管理
- 封禁联动
- 审计日志

### 里程碑结果
形成主站、论坛、游戏服、后台统一运营闭环。

---

## 阶段 5：多游戏扩展与上线准备

### 目标
在首个 PoC 跑通后，开始扩展更多游戏并准备上线。

### 范围
- 第二/第三游戏接入
- 规则扩展
- 风控补强
- 稳定性测试
- 部署与监控

### 里程碑结果
从单游戏 PoC 进入多游戏平台化阶段。
