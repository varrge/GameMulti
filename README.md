# GameMulti

GameMulti 是一个面向多游戏联机社区场景的 Web 平台规划仓库。

当前仓库主要用于沉淀项目的早期规划文档，覆盖：

- 项目总体规划
- 技术选型
- 数据库设计
- 接口边界
- 开发排期
- 任务拆分
- ER 与业务流程说明

## 当前建议方向

- 主站统一身份中心
- 邀请制社区准入
- 游戏内发起绑定，Web 端确认
- 自定义插件接入游戏服务器
- 金币账本与商城兑换分层设计
- Discourse 作为论坛系统首选
